Godot: DataManager - Development Tool

Godot: DataManager offers three objects, DataLoader, DataSaver, and DataManager. DataLoader and DataSaver are dummy workers that do their respective job and are capable of making lossless round-trips between each other in formats: '.bin', '.json', '.csv', and '.cfg'. DataManager combines their capabilities into one package while additionally offering: async loading, data-loading status tracking, smart-single loading, a reversible change log, and a more generalized API. A perfect 'plug and play' database solution.

Godot: TransformEffects - Development Tool and Demonstration

Godot: TransformEffects is an all purpose tool for manipulating the position and rotation of objects within Godot. This allows for rapid implementation of things such as: CPU movement, cinematic camera control, cutscenes, and all sorts of effects. The system delivers a catalogue of pre-implemented effects, but is also open to any custom implementation. The usage of transitions ensures P_0 and C_0 continuity.

The TransformEffects Demonstration is a (non-exhaustive) interactive demonstration of a portion of the pre-implemented effects. Control the camera in the 2D gym or freely control the behavior of 3 cubes in the 3D gym.

Godot: ControllerManager - Development Tool

Godot: ControllerManager keeps a log of every detected device compatible, allowing for a game to keep track of separate player's controllers within the same client. Also provides input management and button configuration, creating consistency for both the player's inputs and the developer's API.

Godot: StateMachine - Development Tool and Demonstration

Godot: StateMachine is a simple framework for creating state machines. This simple framework defines a state machine process that can potentially be the basis for nearly ALL logic within a game.

The StateMachine Demonstration is a minimal 'maze game' where the player decides between a win, lose, and reset button. While the game is played the contents of the game's two state machines are displayed to the player.

Godot: MusicManager - Development Tool

Godot: MusicManager will break tracks into segments, allowing features such as: context-based movement shifts, one-time intros, and free segment and looping control. A rapid BGM solution.
